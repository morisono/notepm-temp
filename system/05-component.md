利用される際には ``\`uml 〜 ``` で囲んでください。

```
package "グループ1"  #blue {
  HTTP - [コンポーネント1]
  [別なコンポーネント]
}

node "グループ2" #red {
  FTP - [コンポーネント2]
  [コンポーネント1] --> FTP
}

cloud {
  [例1]
}


database "MySQL" {
  folder "個人のフォルダー" {
    [フォルダー3]
  }
  frame "別なフレーム" {
    [フォルダー4]
  }
}


[別なコンポーネント] --> [例1]
[例1] ..> [フォルダー3]
[フォルダー3] --> [フォルダー4]

note right of コンポーネント2: 説明です
```
